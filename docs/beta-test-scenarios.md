# 싸와! 베타 테스트 시나리오

## 구매자

### 1. 첫 견적요청

1. 베타 URL 접속
2. 회원가입/로그인
3. 구매자 온보딩 완료
4. 견적요청 작성
5. 품목 3개 추가
6. 납품지역/희망일 입력
7. 견적요청 등록
8. 알림센터 확인

### 2. 견적 비교/선택

1. 도착한 견적 확인
2. 공급업체 정보 확인
3. 가격/배송/세금계산서 조건 비교
4. 업체 선택
5. 거래 생성 확인

### 3. 거래 완료/구매내역

1. 거래 상태 변경 확인
2. 납품 완료 확인
3. 거래 완료 처리
4. 구매내역 생성 확인
5. 장부 반영 대기 확인

### 4. 자료분석

1. 거래명세서 업로드
2. 분석 결과 검토
3. 견적요청으로 변환
4. 구매내역으로 저장

## 공급업체

1. 입점 신청
2. 관리자 승인 후 로그인
3. 매칭 요청 확인
4. 견적 제출
5. 견적 선택 알림 확인
6. 거래 수락
7. 납품 상태 변경
8. 후기/신뢰도 확인

## 관리자

1. 공급업체 승인
2. 전체 견적요청 확인
3. 거래 모니터링
4. 신고/분쟁 처리
5. 피드백 확인
6. 정산/수수료 mock 확인
7. QA 체크리스트 업데이트
