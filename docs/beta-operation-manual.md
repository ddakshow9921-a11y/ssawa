# 싸와! 베타 운영 매뉴얼

## 목적

제한된 구매자/공급업체가 실제 URL로 접속해 견적요청, 견적제출, 거래상태, 자료분석, 피드백 흐름을 검증한다.

## 대상

- 음식점/카페/배달전문점 구매자
- 식자재/포장재/소모품/설비자재 공급업체
- 내부 운영자

## 테스트 가능 기능

- 견적요청 등록
- 공급업체 입점 신청
- 견적 제출/비교/선택
- 거래 상태 관리
- 자료분석 mock
- 구매내역/장부 mock
- 알림/문의/피드백
- 신고/후기/신뢰도 mock

## 제한 기능

- 앱 내 실제 결제 없음
- 실제 에스크로 없음
- 실제 세금계산서 자동 발행 없음
- 실제 홈택스 연동 없음
- 문자/카카오 알림톡 없음
- OCR/AI 결과는 검토용

## 공급업체 승인

1. `/app/admin/suppliers`에서 신청 확인
2. 사업자 정보, 카테고리, 지역, 증빙 확인
3. 승인/보완/반려 처리
4. 승인 후 공급업체에 테스트 요청 배정

## 피드백 처리

1. 사용자가 `/app/feedback` 제출
2. 운영자가 `/app/admin/feedback` 확인
3. 유형/긴급도 분류
4. `submitted -> reviewing -> in_progress -> resolved` 또는 `dismissed`
5. 개발 TODO 또는 이슈로 기록
6. 처리 후 사용자 안내

긴급도:

- urgent: 로그인 불가, 견적요청 불가, 타인 데이터 노출, 관리자 권한 오류
- high: 견적 제출 불가, 파일 업로드 불가, 거래 상태 변경 오류
- normal: UI 깨짐, 문구 오류, 사용성 불편
- low: 개선 제안

## 기본 응대 문구

견적요청:

> 현재 싸와!는 베타 테스트 중이며, 견적 도착 속도와 공급업체 응답 품질을 함께 검증하고 있습니다.

결제:

> 베타 기간에는 실제 앱 내 결제 기능이 제공되지 않으며, 거래 조건은 구매자와 공급업체가 별도로 확인합니다.

분석:

> 자료분석 기능은 현재 테스트 단계이며, 일부 품목은 직접 확인이 필요할 수 있습니다.

오류:

> 오류 화면 또는 상황을 캡처해 보내주시면 우선 확인하겠습니다.
